# Evalgelion

An IRC eval bot

## Support languages

### Python
```
evalgelion: python 1 + 1
```

## Run
```shell
$ git clone https://github.com/openphilippines/evalgelion.git && cd evalgelion
$ sudo docker build -rm -t evalgelion .
$ sudo docker run -d evalgelion
```
